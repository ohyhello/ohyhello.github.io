ticwatch刷国际版系统

#前言
tic最近很伐。疯狂更新新产品却忽略了本质就是用户的体验。系统更新慢而且应用还出的慢。华米快出新品了。人家半个月搞定的事情。出门问问年还没搞好。
##说明
此文章部分翻译和转载加再解释。
刷机需谨慎，否则就又多一块砖头修房子了。
tic国内版的生态真的差。网上流行刷非官方的国际版rom但是这样就会失去nfc 和出门问问自带的功能和携带了许多bug所以本章采用国内rom融合国际版本的方法。

#准备

已解锁的Bootloader
可以正常工作的adb/fastboot和相关驱动
google服务和科学上网（不是必要）
别手滑特别是确定连接线不会松动否则会很烦。

#开冲
##刷入TWRP-Recovery
win+r打开cmd 

adb devices

若出现adb占用问题

adb devices
netstat -ano | findstr "5037"
taskkill /f /pid xxxx 杀死对应进程再重新来
adb reboot bootloader 进入bootloader
fastboot devices
fastboot oem unlock 解锁 bootloader
fastboot boot [twrp-recovery文件名].img

之后会直接进入recovery

1.双清选择Wipe - Factory Rest - 右划确定
2.确定手表与电脑相连，将Conversion patch.zip复制进手表中的TWRP文件夹中
3.在TWRP-Recovery主界面，选择Install - Install Zip - 选择 Conversion patch.zip - 然后点对勾

刷完后重启时候可能还要清一遍 第一次启动时间有点长
要root的就顺便刷入magisk。
重启后就有中国版支持国际版的系统了。
不同的是我在刷recovery的时候发现 adb reboot recovery 不能进入recovery。
